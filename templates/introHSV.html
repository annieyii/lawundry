HSV 色彩分析

HSV = Hue * Saturation * Value

我們系統將一張圖片細分36格，每一格分別計算HSV數值，再用concatenate連接Hue、Saturation、Value三項數值，最終用平均的方式算出整張圖的HSV。

    Hue 色相

    可通過色輪表示。
    0° 為紅色 / 60° 為黃色 / 120° 為綠色 / 180° 為青色 / 240° 為藍色 / 360° 為紅色

    Saturation 飽和度

    為色彩的純度，越高越純粹，越低摻雜越多。
    範圍介於0-1之間，純紅色飽和度為1，反之灰色飽和度接近0。

    Value 明度

    色彩的明亮程度，越高越亮，越低越黑
    範圍介於0-1之間。當明度為0時，呈現黑色。當明度為1時，呈現最亮狀態。
